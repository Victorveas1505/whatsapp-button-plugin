Plugin ligero y fácil de usar que añade un botón flotante de WhatsApp en tu sitio WordPress, permitiendo a tus visitantes contactarte con un solo clic.

== Descripción ==
El Botón Flotante de WhatsApp de Victor Veas te permite conectar con tus clientes o visitantes directamente desde tu sitio web.  
Está diseñado para ser rápido, compatible y muy sencillo de configurar.

**Características:**
- Botón flotante siempre visible en todo el sitio.
- Personalización del número de teléfono.
- Mensaje de bienvenida predefinido.
- Compatible con móviles y escritorio.
- Diseño limpio y moderno.

== Instalación ==
1. Sube la carpeta del plugin a `/wp-content/plugins/`.
2. Activa el plugin en el menú **"Plugins"** de WordPress.
3. Configura tu número de WhatsApp en la página de ajustes del plugin.

== Uso ==
Una vez activado, el botón aparecerá automáticamente en la esquina inferior derecha del sitio.  
Puedes modificar el número y mensaje desde la página de ajustes.
